# BE Svitlo — Home Assistant Integration

Отримує графік відключень по черзі
з https://be-svitlo.oe.if.ua/

## Можливості
- Поточний стан (on/off)
- Наступна зміна
- Сьогоднішні інтервали
- Вибір черги при налаштуванні

## Встановлення через HACS
1. HACS → Integrations → Custom repositories
2. Додати:
   https://github.com/<your-name>/ha-be-svitlo
   тип: Integration
3. Встановити "BE Svitlo"
4. Після встановлення додати інтеграцію через:
   Settings → Devices & Services → Add Integration → BE Svitlo
